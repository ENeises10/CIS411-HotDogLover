This page explains how my CIS 411 MVC assignment was made.

Hot Dog Lover Assignment

This assignment was built in Visual Studio using an ASP.Net application with MVC. After altering the template to fit my needs, I added a model for the project. The model includes four fields that the user will fill out in order to add a new profile or to edit an existing one.

After creating the model, I added a MVC 5 Controller with views, using Entity Framework. I attached it to my newly created model and checked the box to automatically create the views.

Five default views were created: Create, Delete, Details, Edit, and Index. I then modified these views to fit the needs of the assignment. The data is saved using a new DBContext that I created and added to the Web.config file.

I also created a home page to look more like the layout provided by Professor Reisz. The add, edit, and delete functions do not yet properly change the home page, but that will get fixed in the future.

That's how the assignment was made, thanks for reading!
